Web Application

บริการสร้างและออกแบบ Website สำหรับธุรกิจ/กิจการ หรืองานอีเว้นท์ต่างๆ ในรูปแบบ Web Application แสดงผลรองรับทั้งมือถือ แท็ปเล็ต และคอมพิวเตอร์ต่างๆ (Responsive) รวมถึงระบบจัดการข้อมูล (โปรแกรมหลังบ้านต่างๆ) และการดูแลหลังการขาย เพื่อให้ท่านสามารถบริหารจัดการธุรกิจและเข้าถึงลูกค้าได้ง่ายยิ่งขึ้น

Mobile Application

พาธุรกิจของคุณเข้าถึงลูกค้าโดยตรงผ่าน Mobile Application ของคุณเอง ทั้งระบบ iOS และ Android สร้างการจดจำให้กับแบรนด์ นำเสนอโปรโมชั่นด้วย Push Notification รวมถึงวิเคราะห์ข้อมูลของกลุ่มลูกค้าเพื่อพัฒนาธุรกิจ (Report & Statistic)

System Design

ช่วยคุณวางแผนและออกแบบระบบ เพื่อต่อยอด ควบคุมค่าใช้จ่ายในการนำระบบ IT เข้ามาประยุกต์ใช้ให้เกิดประโยชน์มากที่สุด

Data Preparation and Exploration

กระบวนการเตรียมข้อมูล (Data Preparation) คือกระบวนการในการแปลงรูปแบบของข้อมูลดิบ (raw data) ให้อยู่ในรูปแบบที่เหมาะสมก่อนเข้าสู่กระบวนการวิเคราะห์ เป็นขึ้นตอนที่สำคัญซึ่งมักรวมไปถึง กระบวนการจัดรูปแบบของข้อมูล การแปลงข้อมูลให้มีความถูกต้อง การจัดการกับข้อมูลขยะที่อาจปนเปื้อนเข้ามาในระหว่างกระบวนการรวบรวมข้อมูล หรือแม้กระทั่งการรวมข้อมูลดิบจากหลายแหล่งซึ่งมีที่มาและรูปแบบในการจัดเก้บที่แตกต่างกันเพื่อเพิ่มขนาดของฐานข้อมูล ทำให้สามารถวิเคราะห์ข้อมูลเบื้องต้นได้ก่อนเข้าสู่กระบวนการต่อไป

  • เหมาะสำหรับ : ผู้สนใจระดับเริ่มต้นที่มีพื้นฐานด้าน Programming เล็กน้อยหรือผู้ที่ยังไม่มีประสบการณ์ในสายงาน Data Analyse
  • ระยะเวลาเรียน : 1 วัน
  • คอร์สที่ควรเรียนก่อนหน้า : Intro to Python

Basic Machine Learning

กระบวนการ Machine Learning เข้ามามีบทบาทอย่างมากในปัจจุบัน โดยเฉพาะอย่างยิ่งในการประยุกต์ใช้เชิงธุรกิจ และถือเป็นศาสตร์แขนงย่อยในเรื่อง AI ที่สร้างแบบจำลองที่สามารถเรียนรู้และพัฒนาตัวเองได้ โดยไม่จำเป็นต้องป้อนโปรแกรมให้ระบุไว้ โดยกระบวนการเรียนรู้จะเริ่มจากการสังเกตข้อมูลปริมาณมหาศาล เพื่อหารูปแบบและความสัมพันธ์ที่มนุษย์ไม่อาจมองเห็นได้ และนำมาประกอบการตัดสินใจในอนาคตได้อย่างแม่นยำ โดยในภาษา Python นั้นมีเครื่องมีอที่เหมาะสมและใช้งานง่ายสำหรับการสร้าง Machine Learning ซึ่งสามารถนำไปใช้งานจริงได้อย่างกว้างขวาง

  • เหมาะสำหรับ : ผู้สนใจที่มีพื้นฐานด้าน Programming หรือผู้ที่ยังไม่มีประสบการณ์ในสายงาน Data

  • คอร์สที่ควรเรียนก่อนหน้า : Intro to Python, Data Analytics with Python  from Manipulation to Modeling
